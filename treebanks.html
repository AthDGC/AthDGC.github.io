Koine Greek

Period: 3rd century BCE - 4th century CE

Texts: Septuagint, New Testament, Josephus, Philo

Size: 75,000 tokens

Features: Translation alignment, contact phenomena, syntactic change

Byzantine Greek

Period: 4th-15th century CE

Texts: Church Fathers, historical chronicles, legal documents

Size: 60,000 tokens

Features: Medieval syntax, lexical innovation, register variation

Modern Greek

Texts: Early modern literature, religious texts, contemporary works

Size: 40,000 tokens

Features: Modern syntax, diglossia patterns, standardization

Annotation Specifications

PROIEL format standards and features

Morphological Features

  • Part-of-speech tagging
  • Case, number, gender
  • Tense, aspect, mood
  • Person, voice
  • Derivational morphology

Syntactic Features

  • Dependency relations
  • Head-dependent links
  • Syntactic functions
  • Clause structure
  • Argument structure

Metadata

  • Text identification
  • Historical period
  • Genre classification
  • Source information
  • Quality metrics
