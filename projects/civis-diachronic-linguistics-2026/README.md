# Diachronic Linguistics in the 21st Century 2026 - Website

## Overview
This is the official website for the CIVIS Blended Intensive Programme: Diachronic Linguistics in the 21st Century (2026 Edition).

## Website Structure

The website consists of the following pages:

- **index.html** - Homepage with programme overview, learning outcomes, and key information
- **program.html** - Detailed programme structure and schedule (Spring School, Workshops, Summer School)
- **speakers.html** - Faculty profiles (11 professors from CIVIS universities)
- **venue.html** - Information about Naxos, Greece and the summer school facilities
- **registration.html** - Application information, eligibility criteria, and selection process
- **contact.html** - Contact information, FAQs, and useful links
- **style.css** - Modern, responsive stylesheet

## How to Deploy to GitHub Pages

1. **Upload files to your repository:**
   ```bash
   git add .
   git commit -m "Updated website for 2026 edition"
   git push origin main
   ```

2. **Enable GitHub Pages:**
   - Go to repository Settings
   - Navigate to "Pages" section
   - Under "Source", select "Deploy from a branch"
   - Choose "main" branch and "/ (root)" folder
   - Click Save

3. **Your website will be live at:**
   `https://athdgc.github.io/civis-diachronic-linguistics-2026/`

1. Edit the relevant HTML file

3. Colors can be adjusted in the CSS `:root` variables section
4. All images should be placed in the root directory
