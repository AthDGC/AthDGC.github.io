Syntactic Analysis Tools

Advanced tools for dependency parsing and syntactic structure

Dependency Parser

Function: Automatic dependency parsing of PROIEL treebanks

Features:

  • Universal Dependencies compatibility
  • Multiple parsing algorithms
  • Accuracy evaluation metrics
  • Visualization of dependency trees
  • Batch processing capabilities

Input: CoNLL-U format treebanks

Output: Enhanced dependency relations

Argument Structure Analyzer

Function: Analysis of predicate-argument relations

Features:

  • Valency frame extraction
  • Argument role identification
  • Syntactic alternation detection
  • Cross-linguistic comparison
  • Diachronic change tracking

Input: Parsed treebanks

Output: Argument structure patterns

Clause Structure Analyzer

Function: Analysis of clause types and hierarchical structure

Features:

  • Main vs subordinate clause identification
  • Clause boundary detection
  • Embedding depth analysis
  • Coordination patterns
  • Subordination types

Input: Syntactically annotated texts

Output: Clause structure analysis

Constituent Structure Builder

Function: Conversion from dependency to constituent structure

Features:

  • Phrase structure generation
  • Tree format conversion
  • Bracketed notation output
  • Visualization tools
  • Format compatibility

Input: Dependency trees

Output: Phrase structure trees

Morphological Analysis Tools

Tools for morphological pattern recognition and analysis

Morphological Analyzer

Function: Comprehensive morphological analysis

  • POS tag analysis
  • Morphological feature extraction
  • Inflectional paradigm identification
  • Derivational morphology
  • Historical morphology tracking

Lemmatizer

Function: Automatic lemmatization of historical texts

  • Dictionary-based lemmatization
  • Rule-based approaches
  • Statistical lemmatization
  • Historical form recognition
  • Accuracy evaluation

Inflectional Paradigm Builder

Function: Reconstruction of inflectional paradigms

  • Paradigm extraction
  • Pattern recognition
  • Irregular form identification
  • Diachronic paradigm change
  • Cross-linguistic comparison

Translation Analysis Tools

Comparative analysis of translations and source-target relations

Parallel Corpus Analyzer

Function: Analysis of parallel translations

Features:

  • Alignment algorithms
  • Correspondence mapping
  • Translation shift detection
  • Literal vs free translation
  • Quality assessment

Input: Parallel texts in PROIEL format

Output: Alignment and correspondence data

Contact Phenomena Detector

Function: Identification of language contact phenomena

Features:

  • Lexical borrowing detection
  • Syntactic calque identification
  • Morphological transfer
  • Phonological adaptation
  • Contact intensity measurement

Input: Bilingual or multilingual texts

Output: Contact phenomena analysis

Translation Universals Tester

Function: Testing translation universals hypotheses

Features:

  • Explicitation hypothesis
  • Simplification hypothesis
  • Normalization hypothesis
  • Convergence hypothesis
  • Statistical significance testing

Input: Source and target texts

Output: Universal hypothesis testing results

Interlingual Pattern Finder

Function: Discovery of interlingual patterns

Features:

  • Cross-linguistic patterns
  • Translation equivalence classes
  • Pattern generalization
  • Typological classification
  • Pattern frequency analysis

Input: Multiple language pairs

Output: Interlingual pattern database

API and Integration

Programmatic access to analysis tools

REST API

Endpoints:

  • /api/parse - Dependency parsing
  • /api/analyze - Morphological analysis
  • /api/compare - Translation comparison
  • /api/visualize - Data visualization
  • /api/export - Data export

Authentication: OAuth2 required

Rate Limiting: Tier-based access control

Python Library

Installation: pip install athdgc-tools

Features:

  • Local processing capabilities
  • Batch processing
  • Custom analysis pipelines
  • Integration with NLTK/spacy
  • Jupyter notebook support

Documentation: Complete API reference
