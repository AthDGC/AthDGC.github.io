Greek Languages

Complete diachronic development of the Greek language

Classical Greek

Period: 5th-4th century BCE

Characteristics:

  • Complex inflectional morphology
  • Free word order
  • Rich case system
  • Optative mood
  • Participial constructions

Key Texts: Homer, Hesiod, Sophocles, Euripides, Plato, Aristotle

Koine Greek

Period: 3rd century BCE - 4th century CE

Characteristics:

  • Simplified morphology
  • More fixed word order
  • Reduced use of optative
  • Infinitive replacement
  • Septuagint influence

Key Texts: Septuagint, New Testament, Josephus, Philo

Byzantine Greek

Period: 4th-15th century CE

Characteristics:

  • Further simplification
  • Article usage changes
  • Future tense formation
  • Vowel reduction
  • Lexical innovation

Key Texts: Church Fathers, chronicles, legal documents

Modern Greek

Period: 15th century CE - present

Characteristics:

  • Diglossia patterns
  • Katharevousa vs Demotic
  • Simplified case system
  • Periphrastic constructions
  • Standardization efforts

Key Texts: Early modern literature, contemporary works

Germanic Languages

Early Germanic languages with syntactic annotation

Gothic

Period: 4th century CE

Characteristics:

  • Earliest Germanic documentation
  • Strong verb system
  • Complex consonant changes
  • Greek source influence
  • Bishop Wulfila translation

Key Texts: Gothic Bible, Skeireins

Old English

Period: 7th-11th century CE

Characteristics:

  • Germanic base with Celtic influence
  • Latin loanwords
  • Strong/weak verb classes
  • Case system preservation
  • Alliterative poetry

Key Texts: Beowulf, Anglo-Saxon Chronicle

Old Norse

Period: 9th-13th century CE

Characteristics:

  • Conservative Germanic features
  • Four case system
  • Complex verb conjugations
  • Saga literature
  • Skaldic poetry

Key Texts: Poetic Edda, Family Sagas

Italic Languages

Latin and early Romance development

Classical Latin

Period: 3rd century BCE - 2nd century CE

Characteristics:

  • Highly inflected
  • Flexible word order
  • Complex subordinate clauses
  • Rhetorical structures
  • Literary genres

Key Texts: Cicero, Virgil, Ovid, Caesar, Livy

Vulgate Latin

Period: 4th century CE

Characteristics:

  • Simplified syntax
  • Greek influence
  • Biblical vocabulary
  • Translation patterns
  • Medieval Latin basis

Key Texts: Latin Vulgate Bible

Slavic Languages

Early Slavic languages and religious texts

Old Church Slavonic

Period: 9th-11th century CE

Characteristics:

  • First Slavic literary language
  • Complex verbal system
  • Seven case system
  • Greek source influence
  • Cyril and Methodius mission

Key Texts: Biblical translations, liturgical texts

Old Russian

Period: 11th-14th century CE

Characteristics:

  • East Slavic development
  • Church Slavonic influence
  • Case system reduction
  • Chronicle literature
  • Legal code development

Key Texts: Primary Chronicle, Russkaya Pravda
