Prerequisites

Before you start using the AthDGC platform, make sure you have the following:

üîß Technical Requirements
  • Python 3.8+ or Node.js 14+
  • Git for version control
  • Text editor or IDE
  • Internet connection for API access
üë§ Account Requirements
  • Google account for OAuth2 authentication
  • Academic or research affiliation (optional)
  • Valid email address
  • Basic understanding of REST APIs

Installation

Choose the installation method that best fits your needs.

Python Installation

Install the AthDGC Python library using pip:

bash
pip install athdgc

Verify installation:

python
import athdgc print(athdgc.__version__)

JavaScript Installation

Install the AthDGC JavaScript library using npm:

bash
npm install athdgc

Or use yarn:

bash
yarn add athdgc

Command Line Tools

Install the AthDGC CLI tool:

bash
curl -sSL https://raw.githubusercontent.com/athdgc/install.sh | bash

Verify installation:

bash
athdgc --version

Authentication

The AthDGC platform uses OAuth2 authentication for secure API access.

Important: You need a Google account to authenticate with the AthDGC platform.

Step 1: Register for API Access

Visit the API registration page to register your application.

Step 2: Get Your Credentials

After registration, you'll receive:

  • Client ID
  • Client Secret
  • Redirect URI

Step 3: Authenticate

Use the OAuth2 flow to get an access token:

python
from athdgc import Auth # Initialize authentication client auth = Auth( client_id='your_client_id', client_secret='your_client_secret', redirect_uri='http://localhost:8000/callback' ) # Get authorization URL auth_url = auth.get_authorization_url() print(f"Visit: {auth_url}") # After authorization, exchange code for token token = auth.exchange_code_for_token(authorization_code) print(f"Access token: {token.access_token}")

First API Call

Let's make your first API call to list available treebanks.

python
from athdgc import AthDGCClient # Initialize client with your access token client = AthDGCClient(access_token='your_access_token') # List available treebanks treebanks = client.list_treebanks() for treebank in treebanks: print(f"ID: {treebank.id}") print(f"Name: {treebank.name}") print(f"Language: {treebank.language}") print(f"Tokens: {treebank.token_count}") print("---")
javascript
import { AthDGCClient } from 'athdgc'; // Initialize client with your access token const client = new AthDGCClient({ accessToken: 'your_access_token' }); // List available treebanks const treebanks = await client.listTreebanks(); treebanks.forEach(treebank => { console.log(`ID: ${treebank.id}`); console.log(`Name: ${treebank.name}`); console.log(`Language: ${treebank.language}`); console.log(`Tokens: ${treebank.tokenCount}`); console.log('---'); });
bash
curl -X GET "https://api.athdgc.org/v1/treebanks" \ -H "Authorization: Bearer your_access_token" \ -H "Content-Type: application/json"

Basic Usage

Here are some common tasks you'll perform with the AthDGC platform.

Historical Linguistics Research

Analyze diachronic changes in Greek and related languages:

python
# Analyze morphological changes over time results = client.analyze_morphology( text='·ΩÅ ·ºÑŒΩŒ∏œÅœâœÄŒøœÇ ·ºêœÉœÑŒπŒΩ ·ºêŒΩ œÑ·øá ·ºÄŒ≥œÅ·ø∑·æ≥', language='greek', options={ 'include_features': True, 'diachronic_analysis': True } ) for token in results.tokens: print(f"Token: {token.form}") print(f"Lemma: {token.lemma}") print(f"POS: {token.pos}") print(f"Features: {token.features}") print("---")

Corpus Analysis

Search and analyze large linguistic corpora:

python
# Search for specific patterns search_results = client.search( query='·ºÄŒΩŒ∏œÅœâœÄŒøœÇ', treebanks=['greek-classical', 'greek-koine'], search_type='text', options={ 'case_sensitive': False, 'include_context': True } ) print(f"Found {len(search_results)} matches") for result in search_results[:10]: # Show first 10 results print(f"Text: {result.text}") print(f"Treebank: {result.treebank}") print(f"Context: {result.context}") print("---")

Data Export

Export data in various formats for further analysis:

python
# Export treebank data export_data = client.export_treebank( treebank_id='greek-classical', format='json', options={ 'include_morphology': True, 'include_dependencies': True, 'include_metadata': True } ) # Save to file with open('greek-classical.json', 'w') as f: json.dump(export_data, f, indent=2) print("Data exported successfully")
