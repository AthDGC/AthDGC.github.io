Platform Overview

The Athens Digital Glossa Chronos (AthDGC) platform provides comprehensive tools and resources for historical linguistics research, focusing on Greek-Indo-European language contact and diachronic analysis.

üìö Research Tools

Interactive corpus analysis tools for PROIEL treebanks, including morphological analysis, syntactic parsing, and statistical analysis.

  • Morphological analyzer
  • Syntactic parser
  • Dependency visualizer
  • Statistical analyzer
üîå API Access

RESTful API for programmatic access to all platform features and data, with OAuth2 authentication and tiered access control.

  • RESTful endpoints
  • OAuth2 authentication
  • Rate limiting
  • Multiple data formats

Key Features

üåç Multi-Language Support

15+ historical languages across 4 major Indo-European families

üìÖ Diachronic Coverage

Complete coverage from Classical to Modern periods

üîç Advanced Search

Pattern matching, regex support, and complex queries

API Quick Reference

Essential API endpoints for getting started quickly.

GET
/api/v1/treebanks
List all available treebanks
GET
/api/v1/treebanks/{id}
Get specific treebank information
POST
/api/v1/analyze/morphology
Run morphological analysis
POST
/api/v1/analyze/syntax
Run syntactic parsing
Note: All API endpoints require authentication. See our authentication guide for details.

Quick Code Examples

Get started with these basic examples in different programming languages.

Python
import requests # Get treebanks response = requests.get( 'https://api.athdgc.org/v1/treebanks', headers={'Authorization': 'Bearer YOUR_TOKEN'} ) treebanks = response.json() print(treebanks) # Run morphological analysis data = { 'text': '·ΩÅ ·ºÑŒΩŒ∏œÅœâœÄŒøœÇ ·ºêœÉœÑŒπŒΩ', 'language': 'greek' } response = requests.post( 'https://api.athdgc.org/v1/analyze/morphology', json=data, headers={'Authorization': 'Bearer YOUR_TOKEN'} ) result = response.json() print(result)
JavaScript
// Get treebanks const response = await fetch('https://api.athdgc.org/v1/treebanks', { headers: { 'Authorization': 'Bearer YOUR_TOKEN' } }); const treebanks = await response.json(); console.log(treebanks); // Run morphological analysis const data = { text: '·ΩÅ ·ºÑŒΩŒ∏œÅœâœÄŒøœÇ ·ºêœÉœÑŒπŒΩ', language: 'greek' }; const analysisResponse = await fetch('https://api.athdgc.org/v1/analyze/morphology', { method: 'POST', headers: { 'Content-Type': 'application/json', 'Authorization': 'Bearer YOUR_TOKEN' }, body: JSON.stringify(data) }); const result = await analysisResponse.json(); console.log(result);
